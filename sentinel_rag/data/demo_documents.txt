[DOCUMENT 1: Product API Documentation v2.0]
Source: api_documentation_v2.txt
Access Level: 1 | Authority: 0.95

API Rate Limits:
The Sentinel API allows up to 1000 requests per minute for Standard tier accounts.
Enterprise accounts are permitted up to 5000 requests per minute.
Each request must include a valid Bearer token in the Authorization header.

Authentication Flow:
1. Client sends credentials to /auth/token endpoint using POST method.
2. Server validates credentials and returns JWT token (expires in 24 hours).
3. Client includes token in subsequent requests via Authorization: Bearer <token>.
4. Token refresh is available via /auth/refresh up to 5 times before re-authentication.

Error Codes:
- 401: Unauthorized – token missing or invalid
- 429: Too Many Requests – rate limit exceeded
- 503: Service Unavailable – system maintenance window

Pricing Tiers:
- Standard: $99/month, 1000 req/min, 10GB storage
- Professional: $299/month, 3000 req/min, 50GB storage  
- Enterprise: $999/month, 5000 req/min, 500GB storage

The API supports JSON and XML response formats. Default is JSON.
Response times are guaranteed under 200ms for 95th percentile of requests.

How to Reset Your Password:
If you have forgotten your password, navigate to the login page and click "Forgot Password".
Enter your registered email address and click Submit.
You will receive a password reset link within 5 minutes.
The link is valid for 60 minutes. If expired, repeat the process.
Passwords must be at least 12 characters and contain uppercase, lowercase, numbers, and symbols.

Common Issues and Solutions:
Issue: "Cannot connect to server"
Solution: Check your internet connection. Verify the API endpoint URL. Ensure firewall rules allow outbound HTTPS on port 443.

Issue: "Subscription not activating"
Solution: Wait up to 15 minutes after payment for activation. Clear browser cache. Contact support if issue persists after 30 minutes.

Refund Policy:
Customers may request a full refund within 30 days of initial purchase.
After 30 days, refunds are at the discretion of the support team.
Partial refunds are available for unused portions of annual subscriptions.

Support Hours:
Our support team is available Monday through Friday, 9 AM to 6 PM EST.
Priority support for Enterprise customers is available 24/7 via dedicated hotline.

Incident Response Procedures:
In the event of a security breach, the following steps must be taken:
1. Immediately isolate affected systems from the network.
2. Notify the CISO within 1 hour of detection.
3. Preserve all logs and evidence before any remediation.
4. Engage external forensics team if breach involves >10,000 user records.
5. Notify affected users within 72 hours per GDPR requirements.

Encryption Standards:
All data at rest must be encrypted using AES-256.
Data in transit requires TLS 1.3 minimum.
Encryption keys are rotated every 90 days.
Key management system: HashiCorp Vault v1.15+

Penetration Testing:
External penetration tests are conducted quarterly.
The budget allocated for security audits: $250,000 annually.
Last pentest score: 87/100. Three critical vulnerabilities patched in Q3-2024.
Current unresolved low-severity issues: 12 items tracked in security backlog.
